Browser Support

Firefox Accounts is required to work in at least the following environments:

  • Firefox Desktop ESR - 1
  • Firefox for Android ESR - 1
  • Latest versions of modern browsers: Chrome, Safari, Opera, Edge
  • Latest version of Firefox for iOS and iOS Web View
  • Latest version of Firefox for Android and Android WebView
